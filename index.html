Terkadang saya sangat suka membaca namun tak lama saya akan bosan dan lebih memilih menonton video random di YouTube. Kalau ditanya tentang keahlian saya terkadang bingung, karena saya merasa saya bisa melakukan semua hal kalau terdesak, contohnya seperti tugas portofolio ini. Tapi saya senang karena saya jadi bisa belajar html dan css dikit-dikit.
